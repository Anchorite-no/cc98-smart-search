# CC98 Smart Search

一个面向 CC98 的轻量浏览器搜索增强插件。当前小版本采用“原生重排 + API 别名补召回”：让 CC98 先渲染自己的搜索结果，插件移动原有结果卡片；对于 `线性代数` / `线代` 这类简称，再用 CC98 API 补充召回结果并插入同一个列表，不再生成单独的结果面板。

当前 MVP 支持：

- 多关键词搜索
- `+关键词` 必须包含
- `-关键词` 排除
- 基于标题、作者、版面、时间、浏览量的 SearchRank
- API 别名补召回，例如搜 `线性代数` 时补搜 `线代`
- 综合、最新、热门、自定义权重排序
- 点击插件图标开关增强、调节模糊程度、补召回间隔和排序权重

模糊程度会影响别名补召回组数和相关性评分。补召回只查别名查询，不重复查询主关键词；默认会等一小段时间，避开 CC98 原生搜索刚触发的频率窗口。API 失败时会保留 CC98 原生结果，只跳过补召回。

## 安装验证

1. 打开 Chrome：`chrome://extensions/`
2. 开启“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择本项目的 `extension` 目录
5. 打开 `https://www.cc98.org/` 并保持登录
6. 在顶部搜索框输入 `线性代数 期末` 或 `+线代 -答案 期末`
7. 按 Enter 或点击搜索图标

进入 CC98 原生搜索页后，结果列表上方出现 `已按 ... SearchRank 原生重排` 提示即为加载成功。提示中会显示 `API 补召回` 数量；鼠标悬停在结果卡片上可以看到相关性、时间、热度分数。

插件只增强 CC98 的主题/全站/版内搜索。用户搜索和版面搜索暂时交给 CC98 原站处理。
