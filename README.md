# CC98 Smart Search

一个面向 CC98 的轻量浏览器搜索增强插件。当前小版本采用“原生重排”：让 CC98 先渲染自己的搜索结果，插件只移动原有结果卡片，不再额外请求搜索 API，也不再生成一块新的结果面板。

当前 MVP 支持：

- 多关键词搜索
- `+关键词` 必须包含
- `-关键词` 排除
- 基于标题、作者、版面、时间、浏览量的 SearchRank
- 综合、最新、热门、自定义权重排序
- 点击插件图标开关增强、调节模糊程度和排序权重

模糊程度会影响本页原生结果内的别名匹配和相关性评分。为了保持原生重排并避免频繁请求，当前版本不会为了别名额外发起多组搜索。

## 安装验证

1. 打开 Chrome：`chrome://extensions/`
2. 开启“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择本项目的 `extension` 目录
5. 打开 `https://www.cc98.org/` 并保持登录
6. 在顶部搜索框输入 `线性代数 期末` 或 `+线代 -答案 期末`
7. 按 Enter 或点击搜索图标

进入 CC98 原生搜索页后，结果列表上方出现 `已按 ... SearchRank 原生重排` 提示即为加载成功。鼠标悬停在结果卡片上可以看到相关性、时间、热度分数。

插件只增强 CC98 的主题/全站/版内搜索。用户搜索和版面搜索暂时交给 CC98 原站处理。
