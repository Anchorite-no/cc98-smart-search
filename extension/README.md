# CC98 Smart Search MVP

这是 CC98 搜索增强插件的最小版本。

## 功能

- 接管 CC98 顶部主题/全站/版内搜索入口。
- 进入 CC98 原生搜索页后，对原有 `.focus-topic` 结果卡片进行 DOM 重排。
- 对简称/别名使用 CC98 API 补召回，并把补充结果插入同一个原生列表。
- 支持多关键词。
- 支持 `+关键词` 必须包含。
- 支持 `-关键词` 排除。
- 支持小型同义词/简称匹配，用于 API 补召回、本页结果过滤和 SearchRank 评分。
- 提供综合、最新、热门、自定义权重排序。
- 点击插件图标打开设置弹窗，开关增强并调整模糊程度和排序权重。

## 安装

1. 打开 Chrome 扩展管理页：`chrome://extensions/`
2. 开启“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本目录的 `extension` 文件夹。
5. 打开 `https://www.cc98.org/` 并保持登录。

## 使用

在 CC98 顶部搜索框输入：

```text
线性代数 期末
```

或：

```text
+线代 -答案 期末
```

按 Enter 或点击搜索图标后，CC98 会先打开原生搜索页，插件随后在原列表上方显示重排提示，并直接调换原生结果卡片顺序。若模糊程度开启，插件会额外用 API 查询别名，例如 `线性代数` 补查 `线代`，并把新增主题插入同一个列表排序。

## 说明

当前版本只对别名查询请求 `api.cc98.org`，主查询仍交给 CC98 原生搜索页。API 补召回有 8 秒超时和本地缓存；如果 API 被限流或授权失效，页面不会卡住，只展示 CC98 原生结果。

第一版不会增强“用户”和“版面”搜索，这两类搜索仍交给 CC98 原站处理。
