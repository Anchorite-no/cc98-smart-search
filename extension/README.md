# CC98 Smart Search MVP

这是 CC98 搜索增强插件的最小版本。

## 功能

- 接管 CC98 顶部主题/全站/版内搜索入口。
- 进入 CC98 原生搜索页后，对原有 `.focus-topic` 结果卡片进行 DOM 重排。
- 支持多关键词。
- 支持 `+关键词` 必须包含。
- 支持 `-关键词` 排除。
- 支持小型同义词/简称匹配，用于本页结果过滤和 SearchRank 评分。
- 提供综合、最新、热门、自定义权重排序。
- 点击插件图标打开设置弹窗，开关增强并调整模糊程度和排序权重。

## 安装

1. 打开 Chrome 扩展管理页：`chrome://extensions/`
2. 开启“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本目录的 `extension` 文件夹。
5. 打开 `https://www.cc98.org/` 并保持登录。

## 使用

在 CC98 顶部搜索框输入：

```text
线性代数 期末
```

或：

```text
+线代 -答案 期末
```

按 Enter 或点击搜索图标后，CC98 会先打开原生搜索页，插件随后在原列表上方显示重排提示，并直接调换原生结果卡片顺序。

## 说明

当前版本不请求 `api.cc98.org`，因此不会触发额外搜索频率限制。对应代价是：模糊程度只影响原生结果内的过滤和排序，不会额外召回 CC98 原生搜索没有返回的别名结果。

第一版不会增强“用户”和“版面”搜索，这两类搜索仍交给 CC98 原站处理。
