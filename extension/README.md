# CC98 Smart Search MVP

这是 CC98 搜索增强插件的最小版本。

## 功能

- 接管 CC98 顶部主题/全站/版内搜索。
- 支持多关键词。
- 支持 `+关键词` 必须包含。
- 支持 `-关键词` 排除。
- 支持小型同义词/简称扩展。
- 合并多个原站搜索结果并按 SearchRank 排序。
- 提供综合、最新、热门三种排序模式。
- 点击插件图标打开设置弹窗，开关增强并调整模糊程度、请求间隔和排序权重。

## 安装

1. 打开 Chrome 扩展管理页：`chrome://extensions/`
2. 开启“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本目录的 `extension` 文件夹。
5. 打开 `https://www.cc98.org/` 并保持登录。

## 使用

在 CC98 顶部搜索框输入：

```text
线性代数 期末
```

或：

```text
+线代 -答案 期末
```

按 Enter 或点击搜索图标后，插件会在页面中渲染增强搜索结果。

## 说明

主题搜索 API 需要用户登录授权。插件只在 `www.cc98.org` 页面内读取 CC98 自己保存的 `accessToken`，并只用于请求 `api.cc98.org`，不会发送到任何第三方。

第一版不会增强“用户”和“版面”搜索，这两类搜索仍交给 CC98 原站处理。

如果 CC98 返回频率限制提示，建议先把弹窗里的“最多搜索组数”设为 `1`，再逐步调高模糊程度。
