# CC98 Smart Search MVP

这是 CC98 搜索增强插件的最小版本。

## 功能

- 接管 CC98 顶部主题/全站/版内搜索入口。
- 进入 CC98 原生搜索页后，对原有 `.focus-topic` 结果卡片进行 DOM 重排。
- 对简称/别名使用 CC98 API 补召回，并把补充结果插入同一个原生列表。
- 对单字中文短词使用候选短语扩召回、API 返回结果动态挖词和同短语簇降权。
- 支持多关键词。
- 支持轻量中文分词，例如 `线性代数期中韩刚` 会拆成课程、考试、教师关键词。
- 支持 `+关键词` 必须包含。
- 支持 `-关键词` 排除。
- 支持独立别名词库，覆盖 213 组全学科常见课程/学科简称，用于 API 补召回、本页结果过滤和 SearchRank 评分。
- 提供综合、时间优先、热度优先、自定义排序。
- 点击插件图标打开设置弹窗，开关增强并调整模糊程度、补召回间隔；仅在自定义排序时显示权重设置。

## 安装

1. 打开 Chrome 扩展管理页：`chrome://extensions/`
2. 开启“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本目录的 `extension` 文件夹。
5. 打开 `https://www.cc98.org/` 并保持登录。

## 使用

在 CC98 顶部搜索框输入：

```text
线性代数 期末
```

或：

```text
+线代 -答案 期末
```

按 Enter 或点击搜索图标后，CC98 会先打开原生搜索页，插件随后在原列表上方显示重排提示，并直接调换原生结果卡片顺序。若模糊程度开启，插件会额外用 API 查询别名，例如 `线性代数` 补查 `线代`，并把新增主题插入同一个列表排序。

别名词库位于 `lexicon.js` 和 `teacher-lexicon.js`。当前课程别名按学科块维护，覆盖数学、物理化学、生医、计算机、电信、工科、经管、人文社科、地环农建等 213 组常见基础课和专业基础课，例如 `数字逻辑 / 数逻 / digital logic`、`微积分 / vif / 微甲 / 微乙`；老师首字母词库来自 `chalaoshi.de` 详情页 ID 扫描，包含 `9786` 个姓名/首字母强等效组。同一别名组内的词会作为强等效词参与相关性评分和补召回。

短词扩召回不是强等效词库。它只在单字中文查询时触发：先使用 `shortQueryExpansions` 中的候选，例如 `梅花`、`梅雨`、`青梅`，再从当前标题挖包含该字的 2-4 字短语。若词库没有覆盖，例如只搜 `性`，插件会先通过 API 搜原字，并从返回标题中动态挖出候选短语继续补搜；排序时同一短语簇重复出现会被轻微降权。

排序模式：

- 综合：相关性 70% + 时间 15% + 热度 15%
- 时间优先：相关性 40% + 时间 45% + 热度 15%
- 热度优先：相关性 40% + 时间 10% + 热度 50%
- 自定义：使用弹窗中的三个权重滑杆，并按总和归一化

## 说明

当前版本只对别名和短词候选查询请求 `api.cc98.org`，主查询仍交给 CC98 原生搜索页。词库只维护强等效关系，不作为穷举分词表；泛词和未知单字由 API 返回标题动态发现候选。API 补召回默认 1.2 秒后发起，遇到 403 会逐级退避重试，并且有 8 秒超时和本地缓存；如果 API 仍被限流或授权失效，页面不会卡住，只展示 CC98 原生结果。

第一版不会增强“用户”和“版面”搜索，这两类搜索仍交给 CC98 原站处理。
